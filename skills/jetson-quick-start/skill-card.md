## Description: <br>
Entry skill for Jetson / IGX BSP customization that asks one core click-to-select setup questionnaire and passes prefilled answers to downstream setup skills. <br>

This skill is ready for commercial/non-commercial use. <br>

## Owner: NVIDIA <br>

### License/Terms of Use: <br>
Apache 2.0 AND CC-BY-4.0 <br>
## Use Case: <br>
Developers and engineers use this skill to bootstrap a Jetson BSP customization workspace by selecting a setup mode, target platform, BSP release, and custom-carrier intent through a single guided questionnaire, then dispatching the appropriate downstream setup skills. <br>

### Deployment Geography for Use: <br>
Global <br>
